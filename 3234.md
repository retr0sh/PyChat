Инициализация:
Создание главного окна Tkinter
Настройка интерфейса:
	Заголовок "PyChat"
	Поле для ввода имени пользователя
	Окно сообщений (только для чтения)
	Поле ввода сообщений
	Список онлайн-пользователей

Основные функции:
set_user_name() - регистрация имени пользователя
	Проверка на пустое имя или "admin"
	Добавление в список онлайн
	Активация чата

send_massage() - отправка сообщений
	Проверка на пустое сообщение
	Отображение в формате "Имя: сообщение"
	Автопрокрутка к последнему сообщению
